# W3Tools

A collection of web3 development tools and utilities.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

### Building

```bash
pnpm build
```

## Project Structure

- `src/` - Source code directory
- `dist/` - Build output directory
